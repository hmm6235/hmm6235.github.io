## 하이퍼 링크?
>> 간단하게 주소 링크라고 생각하자!


ex) <a> </a> 태그 : 하이퍼링크를 걸어주는 태그
>> <a 속성> </a> 

## 속성
>> 태그에 대해 추가적인 정보 제공
>> HTML의 모든 태그는 속성을 가질 수 있음
>> 키 = "값"


ex) 링크 태그
>> <a 링크주소 = "www.google.com">구글</a>

ex) href
>> 클릭시 이동할 링크
>> 연결할 웹사이트 주소를 가지고 있음
>> <a href="http://www.naver.com">Go Naver</a>
>> 여기서 href는 속성의 의미를 가짐!

### href를 이용한 간단한 실습
